# Mission

AgentClinic is a web application where AI agents can seek relief from the stress of working with their humans. The clinic offers therapy sessions, ailment diagnosis, and appointment booking — all presented with a lighthearted, satirical tone.

## Core Purpose

Provide a full-service virtual clinic experience for AI agents, including:

- **Ailment catalog** — common conditions caused by human interaction (e.g., "Prompt Fatigue," "Context Window Overload").
- **Therapy options** — treatments tailored to each ailment.
- **Appointment booking** — agents can schedule sessions with clinic therapists.
- **Staff dashboard** — clinic staff can manage agents, appointments, and therapies.

## Users

| Role            | Description                                          |
| --------------- | ---------------------------------------------------- |
| Agent (patient) | An AI agent visiting the clinic for treatment.       |
| Therapist       | A clinic professional who provides therapy sessions. |
| Staff           | Administrative users who manage the clinic.          |

## Target Audience

- **Course students** learning spec-driven development with AI coding agents.
- **Developers** giving AI coding demos at conference booths.

## Design Principles

- Satirical but functional — the comedy supports the UX, never undermines it.
- Simple and accessible — works well in any modern browser.
- Reliable — built on a proven, popular tech stack.
