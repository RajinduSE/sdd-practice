# Tech Stack

## Language

- **TypeScript** — used end-to-end (client, server, tooling).

## Framework

- **Next.js** (App Router) — full-stack React framework providing server-side rendering, API routes, and static generation in a single project.

## Frontend

- **React** — component-based UI via Next.js.
- **CSS Modules** or **Tailwind CSS** — scoped, utility-first styling (to be decided in Phase 1).

## Backend

- **Next.js API Routes / Server Actions** — server-side logic co-located with the frontend.
- **Node.js** — runtime environment.

## Database

- **SQLite** — lightweight, file-based relational database. Zero configuration, easy to reset during development and demos.

## Testing

- **Vitest** — fast, Vite-native unit and integration test runner. Used for validation tests across all phases.

## Tooling

- **ESLint** — linting.
- **Prettier** — code formatting.
- **TypeScript Compiler (`tsc`)** — type checking.
